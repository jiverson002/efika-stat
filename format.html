Here is a template for your JSON file Layout

Here is an example of how your JSON file should look

{ "title": "(Your title here)", "xaxis": "(Your xaxis title here)", "yaxis": "(your yaxis title here)", "traces" : [ { "x":["(Date 1)", "(Date 2)"], "y":["(y value here)", "(y value here)"], error_y": { "type": "data", "symmetric": false, "array": [(positive error), (positive error)], "arrayminus": [(negative error), (negative error)] }, "name":"(Name of first line)", "type":"scatter", "text": ["(text on point 1)", "(text on point 2)"] }, { "x":["(Date 1)", "(Date 2)"], "y":["(y value here)", "(y value here)"], error_y": { "type": "data", "symmetric": false, "array": [(positive error), (positive error)], "arrayminus": [(negative error), (negative error)] }, "name":"(Name of second line)", "type":"scatter", "text": ["(text on point 1)", "(text on point 2)"] } ] }

Note: Dates in the JSON file should be formatted as: YYYY-MM-DD
